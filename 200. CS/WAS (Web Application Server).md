### **개념**

- 웹 서버 + 서블릿 컨테이너 + 추가 기능을 포함한 **애플리케이션 실행 환경**
    
- 비즈니스 로직 실행, 트랜잭션, 보안, 연결 풀(Connection Pool) 등  
    **애플리케이션 전반을 관리**
    

### **주요 기능**

- HTTP 요청 처리 (내부적으로 서블릿 컨테이너 사용)
    
- 동적 페이지 처리 (JSP/Servlet, Spring 등)
    
- 트랜잭션 관리, 보안, 리소스 관리
    
- EJB, JPA, JNDI 등 자바 EE 표준 지원
    

### **대표 WAS**

- Tomcat (가벼운 WAS, 서블릿/JSP 중심)
    
- JBoss / WildFly
    
- WebLogic
    
- WebSphere
