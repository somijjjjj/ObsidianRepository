# 📁 Obsidian 폴더 분류 기준


## 📝 폴더 관리 규칙

- 새로운 노트는 일단 `000. Capture`에 생성 → 정리 후 정식 폴더로 이동
- 각 폴더 내에 `README.md`를 두고 그 안에서 세부 분류 작성 가능
- 사용하지 않는 폴더는 2주마다 리뷰하여 삭제 or 통합

## 📁 000~099: 임시 / 개인 메모

- `000. Capture`: 새로 생성한 노트가 자동 저장되는 임시 저장소
  - 주기적으로 정리하여 아래 정식 폴더로 이동
- `001. Note 2025`: 연간 계획이나 목표 노트
