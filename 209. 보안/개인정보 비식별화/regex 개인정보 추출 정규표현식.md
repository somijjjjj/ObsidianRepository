




### 주민등록번호

```
(\d{6}[\s,-]-?[1-4]\d{6})|(\d{6}[\s,-]?[1-4]\d{6})
```

**테스트**
```
991111-1234567
9911111234567
991111 1234567
```


### 운전면허번호

```
(\d{2}-\d{2}-\d{6}-\d{2})
```

**테스트**
```
11-11-111111-11
```
