

## ▫️개인정보


### 주민등록번호

```
(\d{6}[\s,-]-?[1-4]\d{6})|(\d{6}[\s,-]?[1-4]\d{6})
```

**테스트**
```
991111-1234567
9911111234567
991111 1234567
```


### 운전면허번호

```
(\d{2}-\d{2}-\d{6}-\d{2})
```

**테스트**
```
11-11-111111-11
```


### 전화번호/휴대전화번호

```
(0\d{1,2}[\s,.-]-?\d{2,4}[\s,.-]-?\d{4})|(0[0123]{1,2}[\s,.-]?\d{2,4}[\s,.-]?\d{4})
```

```
02-000-0000

010 000 0000
011 111 1111
020000000
031000000

0100000000
0111111111
020000000
031000000
```

```

```

## ▫️민감정보


### 이메일 주소

```
(([\w!-_\.])*@([\w!-_\.])*\.[\w]{2,3})
```

**테스트**
```
a@a.com
bbbb@b.com
a1@a.com
a-a@a.com
abc@abc.com
```

### 나이

```
(\d{0,4}(년생|월생|세|살))
```

테스트
```
80년생
1980년생
3세
10살
```
