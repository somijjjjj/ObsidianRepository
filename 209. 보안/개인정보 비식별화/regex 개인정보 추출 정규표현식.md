

## ▫️개인정보 8종


### 1. 주민등록번호

```
(\d{2})([0]\d|[1][0-2])([0][1-9]|[1-2]\d|[3][0-1])[\s-]?([1-4]\d{6})
```

```
(\d{6}[\s,-]-?[1-4]\d{6})|(\d{6}[\s,-]?[1-4]\d{6})
```

**테스트**
```
991111-1234567
9911111234567
991111 1234567
```


### 2. 여권번호


> 구여권
```
([a-zA-Z]{1}|[a-zA-Z]{2})\d{8}
```
```
([M|S|R|G|D|m|s|r|g|d]{1})(\d{8})
```
> 신여권
```
([a-zA-Z]{1}|[a-zA-Z]{2})(\d{3})([a-zA-Z]{1})(\d{4})
```
```
([M|S|R|G|D|m|s|r|g|d]{1})(\d{3})([a-zA-Z]{1})(\d{4})
```


**테스트**
```
M12345678 //구여권
M123A4567 //신여권
```


### 3. 운전면허번호

```
([1]\d|[2]\d{2})[-](\d{2})[-](\d{6})[-](\d{2})
```

**테스트**
```
11-11-111111-11
11-24-054462-81
```


### 4. 전화번호/휴대전화번호

> 휴대전화번호
```
([01]{2}[0|1|6|7|8|9])[- ]?(\d{3,4})[- ]?(\d{4})
```
> 휴대전화번호+전화번호
```
([01]{2}[0|1|6|7|8|9])[-\s]?(\d{3,4})[-\s]?(\d{4})|([0123]{2,3})[-\s]?(\d{2,3})[-\s]?(\d{4})
```

**테스트**
```
010-0000-0000
010-000-0000
011-111-1111
02-000-0000
031-00-0000
010 0000 0000
010 000 0000
011 111 1111
020000000
031000000
01000000000
0100000000
0111111111
020000000
031000000
```

## ▫️민감정보


### 이메일 주소

```
(([\w!-_\.])*@([\w!-_\.])*\.[\w]{2,3})
```

**테스트**
```
a@a.com
bbbb@b.com
a1@a.com
a-a@a.com
abc@abc.com
```
