

## ▫️개인정보 8종


### 1. 주민등록번호

```
(\d{2})([0]\d|[1][0-2])([0][1-9]|[1-2]\d|[3][0-1])[\s-]?([1-4]\d{6})
```

```
(\d{6}[\s,-]-?[1-4]\d{6})|(\d{6}[\s,-]?[1-4]\d{6})
```

**테스트**
```
991111-1234567
9911111234567
991111 1234567
```


### 2. 여권번호


> 구여권
```
([a-zA-Z]{1}|[a-zA-Z]{2})\d{8}
```
```
([M|S|R|G|D|m|s|r|g|d]{1})(\d{8})
```
> 신여권
```
([a-zA-Z]{1}|[a-zA-Z]{2})(\d{3})([a-zA-Z]{1})(\d{4})
```
```
([M|S|R|G|D|m|s|r|g|d]{1})(\d{3})([a-zA-Z]{1})(\d{4})
```


**테스트**
```
M12345678 //구여권
M123A4567 //신여권
```


### 3. 운전면허번호
